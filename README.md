# Akshara — Learn Telugu Fast

Akshara is an open-source platform that helps children and beginners learn Telugu reading fluently using:

- Flashcards
- Syllables (Guninthalu)
- Word building games
- Sentence reading
- Dictation
- Spaced repetition

## Vision

Help kids read Telugu fluently in 30 days.

## Features

- Telugu flashcards with pronunciation
- Adaptive learning
- Reading speed tracker
- Parent dashboard
- Offline support (PWA)

## Tech Stack

- Next.js
- TypeScript
- TailwindCSS
- Web Speech API (TTS)
- IndexedDB
